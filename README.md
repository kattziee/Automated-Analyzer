Automated AI Data Analyzer

An intelligent, production-grade data analytics platform that automates data cleaning, profiling, visualization, machine learning, and reporting for real-world datasets.


🌐 Live Demo

🔗 Application: [automated-analyzer-shahul.streamlit.app](https://automated-analyzer-shahul.streamlit.app)

🔗 GitHub Repository: https://github.com/kattziee/Automated-Analyzer

📖 Overview

The Automated AI Data Analyzer is an end-to-end analytics platform that enables users to upload datasets and automatically perform:

Data Cleaning
Exploratory Data Analysis (EDA)
Statistical Analysis
Machine Learning
Forecasting
Interactive Visualization
Executive Report Generation

The platform is designed to handle messy, real-world datasets without requiring manual preprocessing.

⚙️ Installation:

Clone Repository

git clone https://github.com/kattziee/Automated-Analyzer.git
cd Automated-Analyzer

Create Virtual Environment

Windows

python -m venv venv
venv\Scripts\activate

Linux/Mac

python3 -m venv venv
source venv/bin/activate

Install Dependencies

pip install -r requirements.txt

Run Application

streamlit run app.py
